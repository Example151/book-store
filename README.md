Requirements:
1. node
2. npm
3. mongoose
4. express
5. body-parser
6. cors
7. dotenv

Steps:
1. enter: npm install
2. create .env file in the root directory and add the following:
    MONGO_URI=your_mongodb_uri
3. enter: npm start
4. open POSTMAN and go to http://localhost:3000/api/books
